# UI 设计终稿 — 气泡宇宙 PC-First 重构

> 参考 Pinterest 瀑布流 + Dribbble 导航/搜索/个人中心设计
> 核心理念：**气泡宇宙** — 每条匿名消息是一个漂浮的气泡
> 日期：2026-03-27

---

## 设计原则

1. **PC 端优先**：1440px 标准设计，不考虑移动端兼容
2. **内容密度适中**：参考 Pinterest 瀑布流但气泡间距更大，强调漂浮感
3. **交互丝滑**：所有状态变化都有过渡动画，hover 是核心交互
4. **气泡即身份**：每个帖子/评论都是独立气泡，有呼吸感

---

## 一、导航栏重构（参考 Dribbble）

**核心变化**：从简单 logo + 图标 → 完整的桌面导航栏

- 高度：**72px**，背景 `rgba(255,255,255,0.8)` + `backdrop-filter: blur(20px)`
- 左侧：Logo（品牌色渐变文字）
- 中间：**搜索栏常驻显示**（不再隐藏在图标后面）
  - 宽度 480px，高度 44px，圆角 22px
  - 背景 `rgba(0,0,0,0.04)`，聚焦时白底 + 品牌色边框 + 阴影展开
  - placeholder 文字："搜索匿名心声..."
  - 左侧搜索图标，右侧清除按钮（有内容时显示）
- 右侧：用户区域
  - 未登录：「登录」「注册」按钮（注册用品牌色实心按钮）
  - 已登录：头像圆形 + hover 弹出下拉菜单（个人中心、我的发布、退出）

**hover 效果**：
- Logo hover 轻微缩放 1.02
- 搜索栏 hover 背景变浅
- 按钮 hover 微浮起 + 阴影

## 二、首页列表页重构（参考 Pinterest）

### 2-1. 顶部区域

- **标签筛选栏**：水平居中，胶囊按钮，间距 12px
  - 未选中：白色背景 + 细边框，hover 时背景变品牌色淡色
  - 选中：品牌色实心 + 白字 + 弹性缩放动画
- **排序/筛选行**：标签栏下方，左排序右时间筛选
  - 排序文字按钮带底部指示条
  - 时间下拉框胶囊样式

### 2-2. 瀑布流气泡网格（核心）

**布局**：CSS `column-count` 瀑布流
- **>=1400px**：5 列（参考 Pinterest 密度但更宽松）
- **1024-1399px**：4 列
- **768-1023px**：3 列
- **column-gap**：28px（比 Pinterest 更大，气泡漂浮感）
- 每个气泡 **margin-bottom: 24px**

**气泡卡片设计**：
- **border-radius: 20px**
- **padding: 20px 24px**
- **box-shadow**: `0 2px 12px rgba(0,0,0,0.06)`
- 背景：保持现有渐变色系统（bubble-1 ~ bubble-8）
- 短文本（<50字）：更紧凑的 padding，更圆润
- 长文本（>100字）：展示 4 行后截断，显示"...展开"

**hover 效果**（参考 Pinterest 的卡片 hover）：
- `transform: translateY(-6px)`
- `box-shadow: 0 12px 36px rgba(0,0,0,0.12)`
- 过渡 300ms ease-out
- 出现半透明操作遮罩层（底部渐变黑→透明），显示点赞/评论按钮

**卡片内容结构**：
- 头部：小头像(32px) + 昵称 + 时间（右对齐）
- 正文：自适应高度
- 底部：标签胶囊（左）+ 点赞/评论计数（右）
  - 操作按钮默认半透明，hover 时完全显示

**入场动画**：
- 使用 `IntersectionObserver`，气泡进入视口时淡入上浮
- 交错延迟：每个气泡延迟 50ms
- `opacity 0 → 1, translateY(20px) → 0, 400ms ease-out`

## 三、详情页重构

**布局**：`max-width: 1000px` 居中，双栏 grid

- 左侧 **60%**：帖子内容大气泡
  - 更大的 padding（28px 32px）
  - 头像放大到 44px
  - 正文字号 16px，行高 28px
  - 底部操作栏：点赞（带动画）、评论数、分享（占位）、删除

- 右侧 **40%**：评论区
  - `position: sticky; top: 88px`，跟随滚动
  - 标题："评论 (N)"
  - 评论列表：每条评论是小气泡
    - 圆角 14px，背景 `rgba(0,0,0,0.03)`
    - hover 时背景加深
    - 子评论左侧 3px 品牌色竖线 + 缩进 24px
  - **底部内联输入框**：
    - 圆角卡片嵌入 textarea
    - 默认单行 44px 高，聚焦时平滑展开到 3 行
    - 右侧发送按钮，品牌色圆角

## 四、发帖页重构

**布局**：`max-width: 960px` 居中，双栏

- 左侧：编辑区
  - 大 textarea（min-height: 280px），半透明背景，聚焦时边框高亮
  - 字数统计（右下角）
  - 标签选择：胶囊按钮组，hover 高亮
  - 背景色选择：色块圆点，hover 放大 1.3x
  - "更多选项"：虚线边框占位按钮（投票、阅后即焚、附件），灰色文字 + "即将上线"

- 右侧：实时预览
  - `position: sticky; top: 88px`
  - 使用选定渐变色渲染真实气泡卡片
  - 随输入实时更新
  - 无内容时显示灰色占位文字

## 五、个人中心重构（参考 Dribbble 个人页）

**布局**：`max-width: 1000px` 居中

### 5-1. 顶部用户信息横幅
- 不用侧边栏，改为**顶部横幅**（参考 Dribbble）
- 左侧：大头像（88px）+ 昵称 + 验证状态 + 注册日期
- 右侧：三个统计数据卡片（发布数 / 获赞数 / 评论数）
  - 数字大号加粗（28px, 700）+ 品牌色
  - 标签小字在下
- 底部：水平 tab 切换条（我的发布 / 我的评论）
  - 选中 tab 底部品牌色指示条 + 弹性滑动动画

### 5-2. 内容区域
- 使用瀑布流布局复用 PostCard
- 3 列，与首页相同的卡片样式
- 空状态：插画/文字提示

---

## 六、全局细节

### 6-1. 背景
- 保持渐变背景 `#F2F0FF → #FFF5F5`
- 3-5 个装饰性漂浮气泡（半透明径向渐变），缓慢漂浮动画
- `prefers-reduced-motion: reduce` 禁用动画

### 6-2. 页面过渡
- 路由切换：淡入 + 上浮（200ms）

### 6-3. 滚动条
- 自定义滚动条样式：细窄（6px），圆角，品牌色 thumb

### 6-4. 发帖 FAB
- 保持右下角悬浮按钮
- hover 时放大 + 阴影加深 + 微旋转（45deg，+ → ×的暗示）
