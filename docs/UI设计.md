# 匿名宇宙 — UI 设计规范（气泡宇宙风格）

## 一、设计理念

**核心隐喻：** 每条匿名消息是一个漂浮的气泡，匿名的心声在宇宙中自由飘荡。

**设计原则：**

- **有机感**：大圆角、柔和渐变、不等高布局，打破方正网格的死板感
- **呼吸感**：大量留白、柔和动效、内容随气泡自然伸缩
- **温柔感**：低饱和渐变色系、圆润字体、无锐角无强阴影
- **沉浸感**：背景微渐变、气泡视差浮动、交互反馈有弹性

| 用途 | 色值 | 说明 |

## 四、间距与栅格

### 4.1 间距基数

以 `4px` 为基础单位，所有间距为 4 的倍数：

| Token | 值 | 常用场景 |
|-------|----|---------|
| `--space-1` | 4px | 图标与文字间距、极小内边距 |
| `--space-2` | 8px | 紧凑元素间距、标签内边距 |
| `--space-3` | 12px | 卡片内元素间距 |
| `--space-4` | 16px | 卡片内边距、列表项间距 |
| `--space-5` | 20px | 区块间距 |
| `--space-6` | 24px | 卡片外间距 |
| `--space-8` | 32px | 大区块分隔 |
| `--space-10` | 40px | 页面级留白 |
| `--space-12` | 48px | 底部安全区域 |

### 4.2 页面边距

| 屏幕宽度 | 水平边距 | 卡片间距 |
|---------|---------|---------|
| < 768px（移动端） | 16px | 12px |
| 768px - 1023px | 24px | 16px |
| ≥ 1024px | 居中容器 960px，auto 边距 | 16px |

### 4.3 布局栅格

- 移动端（< 768px）：单列，卡片全宽
- 平板端（768px - 1023px）：双列瀑布流，等宽
- 桌面端（≥ 1024px）：双列瀑布流，容器最大宽度 960px 居中

| 标签胶囊 | 999px（全圆角） | 标签、筛选项 |
| 头像 | 50%（圆形） | 匿名头像 |
| 图片 | 16px | 帖子内图片 |
| Toast 提示 | 16px | 轻提示 |

## 六、阴影体系

采用柔和扩散阴影，不使用硬阴影：

| 级别 | 值 | 用途 |
|------|----|------|
| Shadow-sm | `0 1px 8px rgba(0,0,0,0.03)` | 默认卡片 |
| Shadow-md | `0 4px 20px rgba(0,0,0,0.06)` | 卡片 hover / 按下 |
| Shadow-lg | `0 8px 32px rgba(0,0,0,0.08)` | 弹窗、浮层 |
| Shadow-float | `0 12px 40px rgba(124,92,252,0.15)` | 悬浮按钮（品牌色阴影） |

深色模式下阴影不可见，改用 `border: 1px solid var(--card-border)` 代替。

## 七、组件设计规范

### 7.1 导航栏（Navbar）

```
┌─────────────────────────────────────────────┐
│  🫧 匿名宇宙          🔍    🔔(3)   👤  │
│  h: 56px                                     │
└─────────────────────────────────────────────┘
```

- 高度：56px（移动端）、64px（桌面端）
- 背景：`var(--card-bg)` + `backdrop-filter: blur(20px)` + `saturate(180%)`
- 底部边框：`1px solid var(--divider)`
- 固定在顶部（`position: sticky; top: 0; z-index: 100`）
- Logo 使用 display 字体 + 气泡 emoji，品牌紫色
- 右侧图标尺寸：24px，间距 20px
- 通知铃铛右上角红点：8px 圆形，品牌辅色，未读数 > 0 时显示
- 搜索图标点击后展开为全宽搜索框（动画 300ms ease-out）

### 7.2 标签筛选栏（Tag Filter Bar）

```
┌─────────────────────────────────────────────┐
│  (全部) (表白💌) (吐槽😤) (求助🆘) (树洞🕳️) ...│
│  h: 48px，横向滚动                            │
└─────────────────────────────────────────────┘
```

- 胶囊形标签：`border-radius: 999px`
- 未选中：`background: var(--card-bg)`，`color: var(--text-secondary)`，`border: 1px solid var(--divider)`
- 选中：`background: var(--brand-primary)`，`color: #FFFFFF`，弹性缩放动画（scale 1 → 1.05 → 1）
- 标签尺寸：padding `8px 16px`，字号 13px，字重 500
- 标签间距：8px
- 容器横向滚动，隐藏滚动条（`-webkit-scrollbar: none`）
- 每个标签可带 emoji 前缀，增加辨识度

### 7.3 排序与时间筛选栏

```
┌─────────────────────────────────────────────┐
│  排序：最新 · 最热 · 推荐    |   今天 ▾      │
│  h: 36px                                     │
└─────────────────────────────────────────────┘
```

- 排序选项：文字按钮组，选中项品牌主色 + 下划线（2px 圆角线，动画滑动）
- 时间筛选：右侧下拉选择器，胶囊形
- 字号 13px，颜色 `var(--text-secondary)`，选中色 `var(--brand-primary)`

### 7.4 帖子气泡卡片（Post Bubble）

**核心组件，需要最精细的设计。**

```
╭─────────────────────────────────╮
│  ┌──┐                           │
│  │头像│ 匿名水母#3927  ·  2小时前 │
│  └──┘                           │
│                                 │
│  帖子正文内容，最多显示3行，超出   │
│  部分截断显示省略号...            │
│                                 │
│  ┌────────┐ ┌────────┐         │
│  │ 图片1   │ │ 图片2   │         │  ← 图片区（可选）
│  └────────┘ └────────┘         │
│                                 │
│  ┌──────────────────────┐      │
│  │ 投票区（可选）         │      │  ← 投票区（可选）
│  └──────────────────────┘      │
│                                 │
│  (表白💌)                        │  ← 标签
│                                 │
│  ♡ 12    💬 5    ☆ 3            │  ← 互动栏
╰─────────────────────────────────╯
```

**尺寸与间距：**
- 卡片内边距：16px
- 圆角：24px
- 卡片间距（瀑布流）：12px
- 卡片背景：帖子选择的渐变背景色（见 2.2 色板）
- 卡片不等高：内容长度决定气泡大小，短文本形成小气泡，长文本展开为大气泡

**头像区域：**
- 头像尺寸：36px × 36px，圆形
- 头像类型：基于用户 ID 生成的渐变色几何图案
- 昵称：13px，字重 600，`var(--text-primary)`
- 时间：12px，`var(--text-secondary)`
- 楼主标识（详情页）：昵称后紧跟胶囊徽章 `[楼主]`，品牌辅色背景白字

**正文区域：**
- 字号：15px，行高 24px
- 卡片列表页最多 3 行，超出 `text-overflow: ellipsis`（使用 `-webkit-line-clamp: 3`）
- 文字颜色根据背景色自动适配：浅色背景用 `#1A1A2E`，深色渐变背景用 `#FFFFFF`（通过计算背景亮度决定）

**图片区域：**
- 图片圆角：16px
- 单图：最大宽度 100%，最大高度 200px，`object-fit: cover`
- 多图（2-3 张）：横向排列，等分宽度，间距 8px，高度 120px
- 点击图片进入全屏预览（Vant ImagePreview）

**标签：**
- 位于互动栏上方
- 胶囊形，padding `4px 10px`，字号 11px
- 背景 `rgba(0,0,0,0.06)`（亮色）或 `rgba(255,255,255,0.1)`（深色）
- 带 emoji 前缀

**互动栏：**
- 底部，水平排列
- 图标尺寸 18px，文字 12px，间距 24px
- 点赞：空心 ♡ → 实心 ♥，激活色 `--brand-secondary`，点击动画（心形弹跳 scale 1 → 1.3 → 1）
- 评论：💬 图标 + 计数，点击跳转详情页
- 收藏：空心 ☆ → 实心 ★，激活色 `--color-warning`

### 7.5 置顶帖气泡

- 尺寸：全宽，比普通卡片更大
- 左上角 📌 置顶标签：品牌主色背景 + 白字胶囊 `[📌 置顶]`
- 边框：`2px solid var(--brand-primary)` + 柔和脉动动画（`box-shadow` 呼吸闪烁，2s 循环）
- 内容不截断，完整展示

## 八、动效规范

### 8.1 缓动函数

| 名称 | 值 | 用途 |
|------|----|------|
| ease-standard | `cubic-bezier(0.4, 0, 0.2, 1)` | 通用过渡 |
| ease-enter | `cubic-bezier(0, 0, 0.2, 1)` | 元素进入 |
| ease-exit | `cubic-bezier(0.4, 0, 1, 1)` | 元素退出 |
| ease-bounce | `cubic-bezier(0.34, 1.56, 0.64, 1)` | 弹性效果（点赞、按钮反馈） |

### 8.2 时长规范

| 级别 | 时长 | 用途 |
|------|------|------|
| Instant | 100ms | 颜色变化、透明度切换 |
| Fast | 200ms | 按钮反馈、图标状态切换 |
| Normal | 300ms | 页面过渡、抽屉弹出、卡片展开 |
| Slow | 500ms | 复杂动画、气泡浮动 |

### 8.3 关键交互动效

| 交互 | 动效描述 | 参数 |
|------|---------|------|
| **点赞** | 心形图标弹跳放大 → 回弹，同时散发 3-5 个粉色粒子 | scale 1→1.4→1, 300ms, ease-bounce |
| **收藏** | 星形旋转亮起 | rotate 0→72°, scale 1→1.2→1, 200ms |
| **卡片加载** | 气泡从下方淡入浮上 | translateY(20px)→0, opacity 0→1, 300ms, 逐个延迟 50ms |
| **页面切换** | 整体淡入 + 轻微上移 | opacity 0→1, translateY(8px)→0, 200ms |
| **下拉刷新** | 顶部气泡膨胀动画 | 自定义刷新指示器，圆形从小变大 |
| **标签切换** | 选中胶囊弹性放大 | scale 1→1.05→1, 200ms, ease-bounce |
| **发帖按钮** | 点击缩小弹回 | scale 1→0.9→1.05→1, 300ms |
| **删除帖子** | 气泡缩小消失 + 透明度降低 | scale 1→0.8→0, opacity 1→0, 300ms |
| **气泡视差** | 页面滚动时卡片有微弱的纵向偏移差异 | 滚动速度 × 0.02 偏移，requestAnimationFrame |

### 8.4 性能约束

- 所有动画仅使用 `transform` 和 `opacity`，避免触发布局重排
- 视差效果使用 `will-change: transform` 提前声明
- 移动端低端设备（`prefers-reduced-motion: reduce`）禁用视差和粒子动效
- 气泡加载动画最多同时运行 6 个（视口内可见卡片数）

## 九、图标规范

### 9.1 图标方案

使用 **线性图标**（Outline），风格圆润、线条均匀（2px 描边），与气泡圆角主题一致。

推荐图标库：[Phosphor Icons](https://phosphoricons.com/)（支持 Vue 组件），选择 `light` 或 `regular` 权重。

### 9.2 图标尺寸

| 场景 | 尺寸 | 说明 |
|------|------|------|
| 导航栏图标 | 24px | 搜索、通知、用户 |
| 互动栏图标 | 18px | 点赞、评论、收藏、分享 |
| 列表项前缀 | 22px | 个人中心菜单 |
| 按钮内图标 | 18px | 与文字并排 |
| 空状态图标 | 64px | 无内容时的占位图标 |

### 9.3 匿名头像生成

采用 **渐变色圆形 + 抽象几何图案** 方案：

- 基于用户 ID 的哈希值生成
- 背景：从预设渐变色池中选择（8 种渐变，映射到 ID 低 3 位）
- 前景图案：3×3 对称点阵，根据 ID 中间位生成（类似 GitHub identicon 但更圆润）
- 图案颜色：白色半透明（`rgba(255,255,255,0.6)`）
- 尺寸：生成 128px PNG，使用时按需缩放

| 通知为空 | 🔔 | "暂时没有新通知" |
| 私信为空 | 💬 | "还没有私信会话" |

### 10.2 加载态

- **骨架屏**：气泡卡片形状的骨架，圆角 24px，内部圆角条纹模拟文字行

- 网络错误：显示断裂的气泡图标 + "网络好像断开了" + [重试] 按钮
- 服务端错误：显示叹号气泡 + "服务器开小差了" + [重试] 按钮
- 错误提示 Toast：红色背景圆角条，顶部弹出，3s 自动消失

## 十二、深色模式特殊处理

### 12.1 帖子背景色适配

深色模式下帖子气泡背景色需要降低饱和度和亮度，保证文字可读性：

```javascript
// 深色模式下的背景色转换逻辑
function adjustColorForDark(hsl) {
  return {
    h: hsl.h,
    s: hsl.s * 0.5,    // 饱和度降至 50%
    l: hsl.l * 0.35     // 亮度降至 35%
  }
}
```

### 12.2 其他深色适配

| 元素 | 亮色 | 深色 |
|------|------|------|
| 卡片阴影 | box-shadow 柔和阴影 | 替换为 border 微弱边框 |
| 图片 | 原样 | 原样，不做处理 |
| 分享卡片 | 亮色方案 | 仍使用亮色方案（分享到外部） |
| 品牌色 | `#7C5CFC` | `#A78BFA`（提亮） |
| 骨架屏 | 灰色系 | 半透明白色系 |
