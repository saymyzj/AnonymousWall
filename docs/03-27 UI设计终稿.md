# 匿名宇宙 — UI 设计终稿（Galaxy 深色宇宙主题）

## 设计方向

**主题名称**：Galaxy（深色宇宙主题）
**核心概念**：气泡宇宙 — 每一个匿名气泡都是星空中的一颗星，半透明玻璃质感 + 彩色辉光 + 星空粒子背景，营造沉浸式的宇宙空间感。

**参考原型**：`ui-prototypes/style-C-galaxy.html`

---

## 一、列表页（首页）设计规范

### 1.1 全局背景

| 项目 | 规范 |
|------|------|
| 底色 | `#0B0D1A`（深邃太空蓝黑色） |
| 星空粒子 | 120 个随机分布的白色圆点（1-3px），闪烁动画（twinkle），`opacity: 0.2 ↔ 0.8`，周期 2-5s 交替 |
| 星云光晕 | 3 个大型渐变圆（300-500px），`filter: blur(100px)`，固定定位，分别为紫色（品牌色 `rgba(124,92,252,0.25)`）、粉色（`rgba(255,107,157,0.2)`）、青绿色（`rgba(6,214,160,0.08)`） |
| z-index 层级 | 星空/星云 z-index: 0，内容 z-index: 1，导航 z-index: 100 |

### 1.2 导航栏

| 项目 | 规范 |
|------|------|
| 高度 | 64px，固定顶部 |
| 背景 | `rgba(11,13,26,0.7)` + `backdrop-filter: blur(24px) saturate(1.5)` |
| 底边框 | `1px solid rgba(255,255,255,0.05)` |
| **左侧** | Logo：🫧 emoji + "匿名宇宙" 渐变文字（brand → pink） |
| **中央** | 胶囊分组 Tab：`发现 / 热门 / 推荐`，包裹在 `rgba(255,255,255,0.06)` 圆角容器中，active 状态为品牌色填充 |
| **右侧** | 通知铃铛（带红点）、消息、用户头像（渐变圆形） |
| **搜索框联动** | 当 Hero 区搜索框滚出视口后，导航栏中央 Tab 区域自动切换为搜索框（带过渡动画），搜索框回到视口后恢复 Tab |

#### 搜索框滚动联动交互细节

```
页面顶部 → Hero 搜索框可见 → 导航栏显示 Tab（发现/热门/推荐）
     ↓ 向下滚动
Hero 搜索框离开视口 → 导航栏 Tab 淡出，搜索框淡入（transition: 0.3s ease）
     ↓ 向上滚动回顶部
Hero 搜索框重新进入视口 → 导航栏搜索框淡出，Tab 淡入
```

- 使用 `IntersectionObserver` 监听 Hero 搜索框的可见性
- 导航栏搜索框样式：`rgba(255,255,255,0.06)` 背景 + `1px solid rgba(255,255,255,0.1)` 边框 + pill 圆角
- focus 状态：边框变为品牌色 + `box-shadow: 0 0 0 4px rgba(124,92,252,0.25)`
