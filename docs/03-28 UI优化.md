# 03-28 UI 优化方案

## 一、优化总览

本次优化以 profile-D-hero-bento.html 和 detail-B-immersive.html 为参考原型，对全站前端 UI 进行统一美化升级。

---

## 二、具体优化项

### 1. 全局星空背景增强

- **目标**：所有页面拥有 profile-D 同款的星光点点效果
- **方案**：
  - 增加星星数量至 180 颗（原 120 颗）
  - 增加流星 (shooting star) 动画效果，随机出现
  - 星星增加微弱的辉光 (glow) 效果
  - 星云 (nebula) 色块增强，增加脉动动画

### 2. 浅色模式修复

- **问题**：切换浅色模式后内容看不清
- **方案**：
  - 浅色模式下 bubble 系列背景色加深不透明度，确保卡片可见
  - 浅色模式下 glass 相关变量调整为半透明白色 + 更深的边框
  - 浅色模式下星空效果调整：星星改为深色粒子，星云改为浅色柔和色块
  - navbar 浅色模式下确保对比度
  - 所有 scoped 样式中硬编码的 `rgba(255,255,255,...)` 需改用 CSS 变量适配

### 3. 个人中心页重设计（参考 profile-D）

- **Hero 区域**：
  - 更大的头像环（渐变光环 + 辉光阴影）
  - 统计数据用独立卡片展示（非内联文字）
  - 加入 hero-kicker 标签（加入时间等信息）
- **Bento 布局**：
  - 保持三栏 bento 网格，面板增加毛玻璃背景 + 圆角阴影
  - Tab 切换增加平滑过渡动画

### 4. 帖子详情页重设计（参考 detail-B）

- **帖子主体**：
  - 去掉单一大色块包裹的"cover"感，改为更精致的卡片式布局
  - 帖子内容区增加左侧装饰条 (accent bar)
  - 标签和作者信息分离展示，更有层次感
- **评论区优化**：
  - 参考 detail-B 的评论区风格，评论间有更清晰的分隔
  - 评论悬停效果增强
  - 嵌套回复的缩进线条使用渐变色
- **去掉返回按钮**：所有页面不再显示自定义返回按钮，依赖浏览器原生后退

### 5. FAB 按钮仅限首页

- **方案**：将 FAB（发布新帖）按钮的显示条件限制为仅在首页 (Home) 路由下展示

### 6. 全局 Toast 通知系统

- **问题**：用户发帖、评论、点赞、收藏、举报等操作缺少统一反馈
- **方案**：
  - 新增 `useToast` composable + `ToastContainer` 组件
  - 支持 success / error / info / warning 四种类型
  - 从屏幕顶部弹出，自动消失（3 秒），带进入/退出动画
  - 替换原有的 actionNotice 内联提示

### 7. 动效增强

- **页面切换**：router-view 增加淡入+上滑动画
- **卡片入场**：帖子卡片、评论列表采用交错 (stagger) 淡入动画
- **按钮交互**：点赞/收藏按钮点击时增加弹跳 (bounce) 微动画
- **Tab 切换**：Profile 页 tab 切换增加内容淡入淡出
- **Hover 效果**：所有可交互卡片增加微上浮 + 发光边框效果

---

## 三、补充优化

- **评论输入框**：focus 时增加发光边框效果（参考 detail-B）
- **空状态**：空列表增加更友好的插画/图标提示
- **加载动画**：优化 spinner 为品牌色脉冲动画

---

## 四、不涉及修改的部分

- 后端 API 接口不变
- 路由结构不变
- 管理后台不在本次优化范围内
